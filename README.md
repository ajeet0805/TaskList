# NODEJS + Express + TypeScript
TaskList – Full Stack Application 
This project contains two repositories: 
----------------------------------------------------------------------------
1. Main Repository (Frontend) 
   React + TypeScript 
   Repo URL: https://github.com/ajeet0805/TaskList.git  
------------------------------------------------------------------------------
2. dev Repository (Backend) 
   Node.js + Express + TypeScript
   ---------------------------------------------------------------------------
Frontend Setup (React + TypeScript) 
Repository 
Clone the main repository: 
  git clone https://github.com/ajeet0805/TaskList.git 
  cd TaskList 

Requirements 
- Node.js (LTS recommended) 
- npm (comes with Node) 
Step 1: Install Dependencies
  npm install 
Step 2: Start Development Server 
  npm run dev
  -------------------------------------------------------------------------------
Backend Setup (Node.js + Express + TypeScript) 
Repository 

Clone or checkout the dev backend repo: 
  git clone <your-dev-repo-url> 
  cd <dev-repo-folder> 

Requirements 
- Node.js 
- npm 

Step 1: Install Dependencies 
  npm install 

Step 2: Start Backend Server 
  npx ts-node server.ts
